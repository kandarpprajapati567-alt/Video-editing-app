Design System & Screen Notes

01
Splash / Onboarding

Brand identity moment. Lists active AI models (Claude, Gemini, etc.) as trust signals. GitHub sign-in ties into your stack.

02
Home Dashboard

Live AI engine status, 4 core tool cards (VFX, SFX, BG Remove, Color), recent project history with processing state.

03
Video Editor

Canvas preview with live effect overlays, timeline scrubber, FX chip selector, and inline AI prompt for natural-language edits.

04
AI Agent Chat

Agentic workflow UI — AI proposes action cards the user can approve and apply. Model switcher shows which AI is active.
